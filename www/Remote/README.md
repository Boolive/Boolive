# Файлы внешних объектов

Если прототипировать от объекта с другого сайта, то файлы класса и значения внешнего объекта будут загружены в эту папку.

У классов автоматически замениться namespace и use на новый. К именам классов добавляется адрес сайта (нормализованный), с которого класс был загружен. Этим обеспечивается уникальность класса.
